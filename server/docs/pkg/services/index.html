type AuthService ¶

AuthService предоставляет методы для аутентификации пользователей. 

type AuthService struct {
}

func NewAuthService ¶ 

func NewAuthService() *AuthService

NewAuthService создает новый экземпляр AuthService.

Возвращает указатель на AuthService.

Пример использования: 

authService := NewAuthService()

func (*AuthService) Auth ¶ 

func (as *AuthService) Auth(username, password string) error

Auth проверяет аутентификацию пользователя.

Принимает имя пользователя и пароль в качестве параметров.

Возвращает ошибку, если имя пользователя или пароль неверны.

Пример использования: 

err := authService.Auth("root", "root")
if err != nil {
	fmt.Println(err)
}

func (*AuthService) Register ¶ 

func (as *AuthService) Register(username, password string) error

Register регистрирует нового пользователя

Принимает имя пользователя и пароль в качестве параметров.

Возвращает ошибку, если имя пользователя уже существует.

func (*ChatService) CreateConversation ¶ 

func (cs *ChatService) CreateConversation(userId string) error

CreateConversation создает новую беседу для пользователя. Параметры:

  • userId: идентификатор пользователя.

Возвращает:

  • error: ошибка, если произошла ошибка при создании беседы.

func (*ChatService) GetMessages ¶ 

func (cs *ChatService) GetMessages(userId, conversationId string) error

GetMessages возвращает все сообщения из беседы для пользователя. Параметры:

  • userId: идентификатор пользователя.
  • conversationId: идентификатор беседы.

Возвращает:

  • error: ошибка, если произошла ошибка при получении сообщений.

func (*ChatService) Send ¶ 

func (cs *ChatService) Send(from, msg string) error

Send отправляет сообщение от пользователя. Параметры:

  • from: идентификатор отправителя.
  • msg: текст сообщения.

Возвращает:

  • error: ошибка, если произошла ошибка при отправке сообщения.
